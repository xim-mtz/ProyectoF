# Aprendizajes del curso Git / GitHub

Durante este curso aprendí a utilizar herramientas y flujos de trabajo que se usan en proyectos reales de desarrollo de software.

### ✅ Principales aprendizajes
1. Manejar el flujo básico de Git: `add`, `commit`, `branch`, `merge`, `push`.
2. Resolver conflictos de manera correcta antes de hacer merge.
3. Contribuir en proyectos de otras personas mediante forks y Pull Requests.
